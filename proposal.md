# Final_Project_Asteroids

## Repository
https://github.com/TroubleN/Final_Project.git 

## Description
This project will produce a window that will let the user play the game of Asteroids from 1979.
This is relevent by help the user/creator to experience playing/creating a video game using only python.

## Features
- Festure 1 - Starting the game:
	- Code will produce a window, fullscreen or windowed, that will play the game immedietly.
- Feature 2 - Control of Player Character:
	- Lets the player controller the characters, movement/aim, and let the player shoot the asteroids.
- Feature 3 - Scoring System:
	- Code will keep track of each destroyed asteroid and adds it to the scoring system displayed on the screen. 

## Challenges
- Will need to learn how to make a complex player controller
- Will need to learn how to take controller of a complex character
- Will need to learn how to make or add sprites
- Will need to learn how to make the asteroids appearing from off screen be able to be destroyed, turn into smaller asteroids, and come in different shapes and sizes

## Outcomes
Ideal Outcome:
- Without issues, create a window that will let the player controll a character that can shoot projectiles that can destroy asteroids that flies towards the character.
- If taken enough damage from asteroids, the player will die and a game over screen will play.
- Scoring system will accuratelly tally the players score for destroying asteroids.
- Adding a new feature from the base game of Asteroids, i.e.: new player character, enemy characters, etc.

Minimal Viable Outcome:
- Produce the base game of Asteroids, with as few of the features uneffective
